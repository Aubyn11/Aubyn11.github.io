矩阵转换向量

假设 v→\overrightarrow{v}v = (1,3,-4)

可写为：

v→\overrightarrow{v}v = [13−4]\begin{bmatrix} 1\\ 3\\ -4\end{bmatrix}⎣⎢⎡​13−4​⎦⎥⎤​ = [100]\begin{bmatrix} 1\\ 0\\ 0\end{bmatrix}⎣⎢⎡​100​⎦⎥⎤​ + [030]\begin{bmatrix} 0\\ 3\\ 0\end{bmatrix}⎣⎢⎡​030​⎦⎥⎤​ + [00−4]\begin{bmatrix} 0\\ 0\\ -4\end{bmatrix}⎣⎢⎡​00−4​⎦⎥⎤​ = 1 ×\times× [100]\begin{bmatrix} 1\\ 0\\ 0\end{bmatrix}⎣⎢⎡​100​⎦⎥⎤​ + 3 ×\times× [010]\begin{bmatrix} 0\\ 1\\ 0\end{bmatrix}⎣⎢⎡​010​⎦⎥⎤​ + (-4) ×\times× [001]\begin{bmatrix} 0\\ 0\\ 1\end{bmatrix}⎣⎢⎡​001​⎦⎥⎤​

所以可得出结论：

v→\overrightarrow{v}v = [xyz]\begin{bmatrix} x\\ y\\ z\end{bmatrix}⎣⎢⎡​xyz​⎦⎥⎤​ = x ×\times× [100]\begin{bmatrix} 1\\ 0\\ 0\end{bmatrix}⎣⎢⎡​100​⎦⎥⎤​ + y ×\times× [010]\begin{bmatrix} 0\\ 1\\ 0\end{bmatrix}⎣⎢⎡​010​⎦⎥⎤​ + z ×\times× [001]\begin{bmatrix} 0\\ 0\\ 1\end{bmatrix}⎣⎢⎡​001​⎦⎥⎤​

所以可得公式：

v→=xp→+yq→+zr→\overrightarrow{v} = x\overrightarrow{p} + y\overrightarrow{q} + z\overrightarrow{r}v=xp​+yq​+zr

其中，v→\overrightarrow{v}v 被表示成 p→,q→,r→\overrightarrow{p},\overrightarrow{q},\overrightarrow{r}p​,q​,r 三个基向量的线性变换
